# NodeAPI - Projeto de Estudos

Este é um projeto prático para estudar o desenvolvimento de APIs utilizando Node.js e Express.

## 🚀 Como começar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor:
   ```bash
   node src/index.js
   ```

3. O servidor estará rodando em `http://localhost:3000`.
